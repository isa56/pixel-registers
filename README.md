## Como rodar a aplicação :arrow_forward:

No terminal, clone o projeto: 

```
git clone [https://github.com/React-Bootcamp-WoMarkersCode/certificate-generator](https://github.com/isa56/pixel-registers)
```

Navegue à pasta clonada e instale as dependências:

```
npm install
```
